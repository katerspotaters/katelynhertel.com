Blocipedia

Blocipedia was the second project while studying at Bloc and it was made to mimic Wikipedia. Through user authorization and authentication, there are two types of users; standard and premium.

Standard users are able to view, create, edit and destroy public wikis. Standard users may upgrade their account using Stripe integration (the main picture) in order to become a Premium user. Premium users can view, create, edit, and destroy both public and private wikis as well as collaborate with other users. They also have to option to downgrade their accounts back to Standard. Blocipedia was created using Ruby on Rails, HTML, CSS, and a little JavaScript and jQuery
